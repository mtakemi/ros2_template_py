# ros2_template_py


## ビルド

```bash
cd ~/unitree_ws
colcon build --packages-select ros2_template_py
source install/setup.bash
```


## 実行方法

```bash

ros2 run ros2_template_py simple_publisher

ros2 run ros2_template_py simple_subscriber

```
